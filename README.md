# rsm-starship-game

This project is a solution to the RSM Software Engineering coding challenge.

## Overview
A simple text-based simulation involving starships, fleets, and starbases.
The application demonstrates object-oriented design, clean structure, and extendable logic.

## How to Run
1. Clone the repositry
2. Open in IDE
3. Run the 'Main' class

## Testing
JUnit tests are located in the '/test' directory.
Run tests using IDE or 'mvn test'.

## Design Notes
- Written in Java
- Focus on clean OOP design and readability
- Easily extendable with new game mechanics
